My AWS sysops exam history of 2022

All the exams I did at home, with different operating systems(MacOS, windows10 and windows 11).

1st time - software problem

  • 23/07/2022,MAC OS, after the supervisor told me he started my exam, he exit chat window, but exam didn’t launch successfully. Had to contact Pearson Vue and waited for two days to get vouch for next time exam.

2nd time - hands-on lab practice

  • 6/8/2022, score is 681 with a no lab done. MAC OS
  • multiple choice, multiple response questions are simple, but failed at three hands-on labs.
  • First reason is that I din’t think labs will be hard, so didn’t prepare and practice at all. Only practices other questions.
  • Second reason is some practices exams on Udemy don’t have labs as well, made me think labs are easy.
  • Labs questions include:

    update cloudformation stack,
    create route53 failover for primary(ALB) and secondary(s3 static website),
    create Cloudwatch alarms, metric filters.

  • I have to sign up for another course with lab, and focus on hands-on again. Then get ready for three or two weeks later.

3rd Time - software problem

  • 26/8/2022, Score is 654 without any labs. MAC OS
    1 Three labs are not hard, as long as follow the steps.
  • Web ACL with rules allow CIDR range access or based on path access times.
  • SNS topics, subscription,
  • Cloudwatch alarms, metric filters
  • Create EC2 spot fleet, from EC2 dashboard-> Spot Requests->Request Spot instances-> then choose launch template.
    2 Single or multiple choices, as long as you practice it’s ok to pass.
  • The only not annoying thing is the AWS exam software of Pearson Vue too slow to respond.
    I cannot do my labs properly, and lose patient to finish the test in the end, so failed exam again.

4th time -software problem

  • 10/9/2022, Score is 671 without any labs. Windows 10 Operating system.
  • I used my BF’s computer, because I found the system testing software works better on windows than MACOS. So during the exam, everything is smooth, until I finished multiple and single respond questions and ready for labs, but labs did not load. My supervisor helps to reload software two times and Pearson VUE technical support people let me reboot my computer two times, no magic happens.
  • The exam experience with Pearson Vue is sooooooooo…. frustrating. I didn’t think at that time, I have the energy to try it again.

5th time -pass

  • From previous experience, I tried macos for pearson vue, even system test can feel very slow. Tried on my BF’s game computer which is window os, multiple choices parts are very smooth, but failed at loading labs. So this time, I bought a new Windows Laptop which is only used for online exam test.
  • 22/9/2022, checked in successfully, loading multiple choices success, and even the three labs success, finished exam in 2 hours.
  • And this is the fist time, I read exam carefully, and I saw it says multiple choices takes up of 80% score of the exam, and labs take up of 20%.
  • What I learnt from all these experience, if the software has some problems, don’t struggle to use, ask someone to fix it, if cannot, then do it the next time and let Pearson Vue cancel the failed exam score.
  • Most importantly, never lose your patience!
